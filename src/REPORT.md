# Отчёт о тестировании приложения "Money Transfer"

## Краткое описание

В результате проведённой работы было выяснено, что программа с указанными входными данными из [Задачи №1](https://github.com/netology-code/javaqa-homeworks/tree/master/programming) не выполняет свои функции, т.е. не пополняет счёт клиента на установленную условием задачи сумму.

## Описание тестов

Было проведено функциональное тестирование приложения Money Transfer. Тестировалась функция пополнения счёта клиента. Всего в рамках процесса тестирования был проведён один тест, направленный на проверку работы основной функции приложения.

## Результаты

1. 100% не успешных тестов
2. Баг-репорт: [При пополнении счёта программа выдаёт отрицательное итоговое значение](https://github.com/asloba/java-homeworks2.1/issues/1)

## Общие рекомендации

Рекомендовано использовать другой целочисленный тип данных (long вместо int).
